# Retirement Investment Calculator

## Overview
This Java console application simulates retirement savings growth over time using compound interest. It allows the user to enter financial details and projects yearly balances until retirement.

## Features
- Input validation for all user entries
- Supports annual, monthly, and daily compounding
- Supports annual contribution increases
- Displays a year-by-year projection table
- Calculates total contributions and total interest earned
- Allows multiple simulations in one run

- Current Balance

- Annual Contribution Increase (%)
- Compounding Frequency (Annually, Monthly, Daily)

## Output
- Year-by-year projection showing:
  - Age at end of year
  - Start balance
  - Contributions
  - Interest earned
  - End balance
- Summary section including:
  - Total contributions
  - Total interest earned
  - Ending balance at retirement

## How to Run
1. Open the project in Eclipse.
2. Run `RetirementCalculator.java` as a Java Application.
3. Follow the prompts in the console.
